## ✨ What vsWaybar-Studio does

vsWaybar-Studio is a visual editor for Waybar. It helps you build, style, and preview your bar in real time. You can change layout, colors, spacing, and modules without working through long config files by hand.

It is built with GTK and Python and fits well with Wayland desktop setups. If you use Waybar on Linux and want a clearer way to edit it, this app gives you a simple screen to work from.

## 🧩 What you can do

- Create and edit Waybar layouts
- Change styles with live preview
- Adjust module order and placement
- Work with JSON-based settings
- Tune colors, fonts, and spacing
- Test changes before saving them
- Keep your bar matched to your desktop theme

## 🛠️ Basic workflow

### 1. Open your config
Load the current Waybar config file into the editor. This gives you a starting point and keeps you from editing the wrong file.

### 2. Edit the layout
Choose where each module should sit on the bar. You can move items into left, center, or right sections.

### 3. Change the style
Edit the look of the bar. Many users start with the background and text, then move to spacing and borders.

### 4. Preview the result
Check the bar in real time. This helps you see problems before you save.

### 5. Save and test
Save your changes and reload Waybar to confirm that the new setup works.
